# Libs necessárias

Antes de instalar o mod, você precisa ter alguns mods necessários para seu funcionamento

### Cleo Mod (libCLEOMod.so), versão recomendada 2.0.1.9
Disponível no Discord do AML

### SA Utils (libSAUtils.so), versão recomendada 1.6
Disponível no Discord do AML

### MenuSZK, versão recomendada 1.0
Github: https://github.com/Danilo1301/menu-szk-mobile-releases

# Como instalar

1. Baixe o .zip da versão mais recente em [Releases](https://github.com/Danilo1301/parked-vehicles-mobile/releases)
2. Abra o zip baixado e copie os arquivos de "com.rockstargames.gtasa" para dentro da pasta raíz do seu gta (ex: em Android/com.rockstargames.gtasa/)

# Como usar

Dentro da pasta /mods/mods_data/parkedVehicles/vehicles ficam os grupos de veículos.

Cada .txt define um grupo, por exemplo, "Veículos de LS.txt"

Dentro do .txt você coloca a configuração que define onde os veículos irão spawnar

```
# ID, X, Y, Z, ANGLE
596, 2555.05, -1772.56, 13.56, 90
523, 2555.05, -1772.56, 13.56, 270
416, 2555.05, -1772.56, 13.56, 90
```

ID = id do veículo

X, Y, Z = posição no mundo do GTA

ANGLE = rotação do veículo, entre 0-360

Dica: você pode pegar as coordenadas usando o mod https://github.com/Danilo1301/get-position-mobile
